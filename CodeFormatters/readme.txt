Para colocarem os formatters, têm que fazer:

* Eclipse > Window > Preferences > Java > Code Style > Formatter > Import > 'javaEclipseFormatter' > Apply

* Eclipse > Window > Preferences > JavaScript > Code Style > Formatter > Import > 'angularEclipseFormatter' > Apply

* Para depois formatarem quando estiverem a desenvolver os comandos são: Ctrl + Shift + F.

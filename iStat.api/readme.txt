############## Informações ##############

***** Quinta-feira, 8 de Dezembro de 2016 *****

- Para terem a base de dados operacional, precisam de instalá-la, seguindo o que está no site do MongoDB

- Depois, é preciso executarem numa linha de comandos: mongod --port 9090

- De seguida, noutra linha de comandos: mongo --port 9090
 
- Executar o comando na segunda linha de comandos: use iStat (serve para criar a BD)

- Para verificarem as 'tabelas' existentes, o comando é: show collections

- Para fazerem um select à 'tabela', o comando é: db.<collection>.find(), sendo que a nossa collection tem o nome de DocumentIStat

- Não é preciso criarem as collections ('tabelas'), elas são criadas em runtime.

- Depois de terem tudo isto a correr, podem então experimentar os métodos de Save e Open dataset.
